# Pyline PCBA Metrics Site

This project is a static website designed to display information about PCBA metrics and the software solutions offered by Pyline Solutions, specifically Proline Graphs and Proline Analytics.

## Project Structure

```
pyline-pcba-metrics-site
├── css
│   └── styles.css
├── js
│   └── main.js
├── pages
│   ├── pcba-metrics.html
│   ├── proline-graphs.html
│   └── proline-analytics.html
├── 404.html
├── index.html
└── README.md
```

## Description

- **CSS**: The `css/styles.css` file contains styles for the website, ensuring a visually appealing layout and typography.
  
- **JavaScript**: The `js/main.js` file includes scripts for interactivity, enhancing user experience through dynamic content.

- **PCBA Metrics Page**: The `pages/pcba-metrics.html` file provides detailed insights into PCBA metrics, their significance, and how they contribute to production efficiency.

- **Proline Graphs Page**: The `pages/proline-graphs.html` file describes the features of Proline Graphs, including real-time performance tracking and historical data visualization.

- **Proline Analytics Page**: The `pages/proline-analytics.html` file outlines the capabilities of Proline Analytics, focusing on analyzing component rejection rates and costs.

- **404 Page**: The `404.html` file serves as a custom error page for handling non-existent pages.

- **Homepage**: The `index.html` file acts as the entry point of the website, providing an overview of Pyline Solutions and linking to other pages.

## Getting Started

To set up the website locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/pyline-pcba-metrics-site.git
   ```

2. Navigate to the project directory:
   ```
   cd pyline-pcba-metrics-site
   ```

3. Open `index.html` in your web browser to view the website.

## Deployment

This website can be hosted on GitHub Pages. To deploy:

1. Push your code to a GitHub repository.
2. Go to the repository settings.
3. Under the "Pages" section, select the branch to deploy from (usually `main` or `gh-pages`).
4. Your site will be available at `https://yourusername.github.io/pyline-pcba-metrics-site`.
