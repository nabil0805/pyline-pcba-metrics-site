Proline Analytics

Analytics Modules

Module-based analysis built for action

The software is built around four primary modules that cover overview analytics, placement performance, full event traceability, and recurring PCB failure patterns.

1. Summary

View rejected components, reject quantities, reject classifications, machine usage, unit component cost, and total reject cost per component.

2. Successful Placements

Analyse successfully placed components, placement quantities, board-level placement statistics, component value, and total successful placement cost.

3. Individual Spit Events

Track every reject and spit event with complete context so engineers can investigate exact production events and costs.

4. Repeated Locations

Identify recurring reject locations and repeated placement failures on the PCB to support root-cause investigation.
